● This is a personal finance tracking web application called Spendly, built with
  Flask (Python web framework). It appears to be an educational project where
  students will implement features step-by-step.

  Project Overview

  - Purpose: Help users track expenses, understand spending patterns, and take
  control of their finances
  - Tagline: "Track Every Rupee"
  - Current State: Basic Flask app with landing, login, and register routes
  implemented; other features marked as placeholders for student implementation

  Key Files & Structure

  - app.py: Main Flask application with routes for:
    - Landing page (/)
    - User registration (/register)
    - User login (/login)
    - Placeholder routes for logout, profile, expense management (to be
  implemented)
  - database/db.py: Placeholder for database setup (students will implement
  SQLite connection, table creation, and seeding)
  - templates/: HTML templates using Jinja2:
    - Base layout with navigation and styling
    - Landing page featuring expense tracking benefits and mock visualizations
    - Login and registration pages
  - static/: CSS (style.css) and JavaScript (main.js) for frontend styling and
  interactivity
  - requirements.txt: Flask 3.1.3, Werkzeug, pytest for testing

  Planned Features (from comments in code)

  1. Database setup with SQLite
  2. User authentication (login/logout)
  3. Expense tracking (add, edit, delete expenses)
  4. User profile management
  5. Spending pattern analysis and filtering

  The application uses a clean, modern UI with financial tracking visualizations
  and is designed to be built incrementally through guided steps.
